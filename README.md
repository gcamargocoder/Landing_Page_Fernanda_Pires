<a name="about"></a>
## :house: About the project

This is a **premium, single-page website** built for **Fernanda Pires Camargo**, a TFT (Thought Field Therapy) therapist who treats anxiety, phobias and trauma through individual and group sessions.

The goal was to build a site that feels **trustworthy, professional and warm** вЂ” the kind of first impression that makes a visitor think *"if the website is this good, imagine the care."*

Beyond serving Fernanda's practice, this project was also built as a **reusable premium template**: swap the logo, photos, colors and copy, and it's ready to sell to other therapists and small businesses.

<a name="features"></a>
## вњЁ Features

- 100% **responsive design**, from large desktops down to small phones
- **Scroll-reveal animations** that replay every time a section enters or leaves the viewport
- Sticky **header** that adapts style on scroll
- Mobile **slide-in navigation** menu
- **Testimonials carousel** with autoplay (pauses on hover) and dot navigation
- **FAQ accordion** with full keyboard/screen-reader support (`aria-controls`, `aria-labelledby`)
- Floating **WhatsApp button**
- SEO-ready: Open Graph, Twitter Card, canonical URL, JSON-LD (`ProfessionalService`)
- Performance-minded: preloaded hero image, lazy-loaded below-the-fold images, explicit image dimensions to avoid layout shift
- Accessible: skip-to-content link, labeled landmarks, sufficient color contrast
